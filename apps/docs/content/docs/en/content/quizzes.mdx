Quizzes help you measure student understanding and reinforce learning.

## Create a quiz

1. Go to **Content > Quizzes**
2. Click **New Quiz**
3. Enter a title and description
4. Save

## Add questions

Click **Add Question** and choose the type:

### Multiple choice

One correct answer from several options.

- Enter the question
- Add 2-4 options
- Mark the correct one
- Add an explanation (shown after answering)

### Multiple select

Multiple correct answers are possible.

- Students must select all correct options
- Useful for "select all that apply" questions

### True/False

Simple binary questions.

- Enter a statement
- Mark if it's true or false

## Question tips

- Keep questions clear and concise
- Avoid trick questions
- Write helpful explanations
- Mix question types for variety

## AI generation

You can generate quiz questions automatically from:

- Video transcripts
- Document content

Go to the quiz and click **Generate Questions**. Select the content source and the AI will suggest relevant questions.
