Documents complement your video lessons with downloadable resources.

## Upload a document

1. Go to **Content > Documents**
2. Click **New Document**
3. Enter a title
4. Upload your file

Supported formats: PDF, DOCX.

## Use cases

- **Guides**: Step-by-step instructions
- **Templates**: Downloadable files students can use
- **Summaries**: Key points from video lessons
- **Worksheets**: Exercises to practice

## Text extraction

When you upload a PDF, Learnbase extracts the text content. This allows:

- The AI assistant to answer questions about the document
- Search within document content
- Better context for quiz generation

## Document details

For each document you can set:

- **Title**: The name shown to students
- **Description**: What the document contains
