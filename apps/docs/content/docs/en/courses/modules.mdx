Modules are sections that group related content within a course.

## Create a module

1. Go to **Content > Modules**
2. Click **New Module**
3. Enter a title and description
4. Save

## Add content to a module

Each module can contain:

- **Videos**: Recorded lessons
- **Documents**: PDFs, guides, resources
- **Quizzes**: Assessments to test knowledge

To add content:

1. Open the module
2. Click **Add Item**
3. Select the content type
4. Choose from your library or create new

## Organize content

- Drag and drop to reorder items within a module
- Mark items as "Preview" to let non-enrolled users see them
- Content order determines the learning sequence

## Add modules to a course

1. Go to your course
2. In the **Modules** section, click **Add Module**
3. Select modules from your library
4. Drag to reorder

A course can have multiple modules, and a module can be reused in different courses.
