## Estados do curso

Um curso pode estar:

- **Rascunho**: Visível apenas para administradores. Use isso enquanto constrói o curso.
- **Publicado**: Visível para todos no catálogo. Os estudantes podem se inscrever.

## Antes de publicar

Certifique-se de que seu curso tenha:

- Um título e descrição claros
- Pelo menos um módulo com conteúdo
- Uma imagem de capa
- O preço correto (ou gratuito)

## Publicar seu curso

1. Vá para **Conteúdo > Cursos**
2. Encontre seu curso
3. Mude o status de **Rascunho** para **Publicado**

O curso aparecerá imediatamente no catálogo do seu campus.

## Despublicar

Se precisar fazer alterações ou ocultar temporariamente um curso:

1. Mude o status de volta para **Rascunho**
2. Os estudantes atualmente inscritos manterão seu acesso
3. Novas inscrições não serão possíveis até você publicar novamente
