Os módulos são seções que agrupam conteúdo relacionado dentro de um curso.

## Criar um módulo

1. Vá para **Conteúdo > Módulos**
2. Clique em **Novo Módulo**
3. Digite um título e descrição
4. Salve

## Adicionar conteúdo a um módulo

Cada módulo pode conter:

- **Vídeos**: Aulas gravadas
- **Documentos**: PDFs, guias, recursos
- **Quizzes**: Avaliações para medir conhecimento

Para adicionar conteúdo:

1. Abra o módulo
2. Clique em **Adicionar Item**
3. Selecione o tipo de conteúdo
4. Escolha da sua biblioteca ou crie um novo

## Organizar conteúdo

- Arraste e solte para reordenar itens dentro de um módulo
- Marque itens como "Preview" para que usuários não inscritos possam vê-los
- A ordem do conteúdo determina a sequência de aprendizado

## Adicionar módulos a um curso

1. Vá para seu curso
2. Na seção **Módulos**, clique em **Adicionar Módulo**
3. Selecione módulos da sua biblioteca
4. Arraste para reordenar

Um curso pode ter múltiplos módulos, e um módulo pode ser reutilizado em diferentes cursos.
