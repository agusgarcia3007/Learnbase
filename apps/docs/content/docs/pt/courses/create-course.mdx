## Informações básicas

Vá para **Conteúdo > Cursos** e clique em **Novo Curso**.

Preencha os campos obrigatórios:

- **Título**: O nome do seu curso
- **Descrição**: O que os estudantes aprenderão
- **Categoria**: Como classificá-lo
- **Nível**: Iniciante, Intermediário ou Avançado

## Mídia

Adicione elementos visuais:

- **Capa**: A imagem que aparece no catálogo
- **Vídeo preview**: Um vídeo curto que os estudantes podem ver antes de se inscrever

## Preço

Configure o preço:

- **Preço**: O valor a cobrar
- **Moeda**: USD, EUR ou sua moeda local
- **Preço original**: Se quiser mostrar um desconto

## Detalhes adicionais

Opcionalmente adicione:

- **Características**: O que está incluído (ex: "10 horas de vídeo", "Certificado")
- **Requisitos**: O que os estudantes precisam saber previamente
- **Objetivos**: O que poderão fazer após completar o curso

## Certificado

Habilite a opção de certificado se quiser que os estudantes recebam um certificado ao completar o curso.
