Faça seu campus refletir a identidade da sua marca.

## Logo e favicon

Vá para **Site > Personalização**:

- **Logo**: Aparece no header. Recomendado: PNG com fundo transparente.
- **Favicon**: O pequeno ícone nas abas do navegador. Recomendado: Quadrado, pelo menos 32x32px.

## Tema

Escolha entre temas predefinidos:

- Default
- Slate
- Rose
- Emerald
- Tangerine
- Ocean

Cada tema define cores para botões, links e destaques.

## Cores

Para mais controle, personalize cores individuais:

- **Primária**: Cor de ação principal (botões, links)
- **Fundo**: Fundo da página
- **Texto**: Cor do texto principal

As cores se adaptam automaticamente para modo claro e escuro.

## Tipografia

Escolha suas fontes:

- **Fonte de títulos**: Para títulos
- **Fonte do corpo**: Para texto regular

## Padrões de fundo

Adicione interesse visual às suas páginas:

- Grid
- Dots
- Waves
- Nenhum

Os padrões aparecem sutilmente atrás do conteúdo.

## Pré-visualização

As alterações aparecem em tempo real enquanto você ajusta as configurações. Clique em **Salvar** quando estiver satisfeito com o resultado.
