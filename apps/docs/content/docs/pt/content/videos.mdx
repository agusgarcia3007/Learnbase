Os vídeos são o tipo principal de conteúdo para seus cursos.

## Fazer upload de um vídeo

1. Vá para **Conteúdo > Vídeos**
2. Clique em **Novo Vídeo**
3. Digite um título
4. Clique na zona de upload e selecione seu arquivo

Formatos suportados: MP4, WebM, MOV.

## Transcrição automática

Quando você faz upload de um vídeo, o Learnbase pode automaticamente:

- Gerar uma transcrição do áudio
- Sugerir um título e descrição baseados no conteúdo

Isso ajuda com:

- Acessibilidade para estudantes com problemas auditivos
- Busca dentro do conteúdo do vídeo
- Contexto para o assistente IA

## Detalhes do vídeo

Para cada vídeo você pode configurar:

- **Título**: O nome mostrado aos estudantes
- **Descrição**: Contexto adicional sobre a aula
- **Duração**: Detectada automaticamente do arquivo

## Dicas

- Mantenha os vídeos entre 5-15 minutos para melhor retenção
- Use boa qualidade de áudio (importa mais que a qualidade de vídeo)
- Adicione marcadores de capítulos no seu editor de vídeo para fácil navegação
