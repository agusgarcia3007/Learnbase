Os documentos complementam suas aulas em vídeo com recursos para download.

## Fazer upload de um documento

1. Vá para **Conteúdo > Documentos**
2. Clique em **Novo Documento**
3. Digite um título
4. Faça upload do seu arquivo

Formatos suportados: PDF, DOCX.

## Casos de uso

- **Guias**: Instruções passo a passo
- **Templates**: Arquivos para download que os estudantes podem usar
- **Resumos**: Pontos-chave das aulas em vídeo
- **Planilhas de exercícios**: Exercícios para praticar

## Extração de texto

Quando você faz upload de um PDF, o Learnbase extrai o conteúdo de texto. Isso permite:

- Que o assistente IA responda perguntas sobre o documento
- Busca dentro do conteúdo do documento
- Melhor contexto para geração de quizzes

## Detalhes do documento

Para cada documento você pode configurar:

- **Título**: O nome mostrado aos estudantes
- **Descrição**: O que o documento contém
