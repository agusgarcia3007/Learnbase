## Como funciona a inscrição

Os estudantes podem se inscrever em cursos de duas formas:

### Auto-inscrição

Os estudantes navegam pelo seu catálogo, selecionam um curso e se inscrevem. Se o curso for pago, completam o checkout primeiro.

### Inscrição manual

Os administradores podem inscrever estudantes diretamente:

1. Vá para **Gestão > Inscrições**
2. Clique em **Nova Inscrição**
3. Selecione o estudante e o curso
4. Salve

## Ver inscrições

Vá para **Gestão > Inscrições** para ver:

- Todas as inscrições no seu campus
- Filtrar por curso ou estudante
- Data de inscrição e progresso

## Remover inscrição

Se precisar remover um estudante de um curso:

1. Encontre a inscrição
2. Clique no botão excluir
3. Confirme

O estudante perderá acesso ao conteúdo do curso mas seus dados de progresso são mantidos.

## Inscrição em lote

Para adicionar múltiplos estudantes de uma vez:

1. Use a função do carrinho de compras
2. Adicione múltiplos cursos
3. Complete o checkout para se inscrever em todos de uma vez
