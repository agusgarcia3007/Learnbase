## Como funcionam os certificados

Quando um estudante completa 100% de um curso com certificados habilitados, um certificado é gerado automaticamente.

O certificado inclui:

- Nome do estudante
- Nome do curso
- Data de conclusão
- Código de verificação único
- Código QR para verificação

## Habilitar certificados

Para cada curso:

1. Vá para **Conteúdo > Cursos**
2. Edite o curso
3. Habilite **Incluir Certificado**
4. Salve

## Personalizar certificados

Vá para **Site > Configuração** para personalizar:

- **Assinatura**: Faça upload da sua imagem de assinatura
- **Nome do assinante**: Quem assina o certificado
- **Mensagem personalizada**: Texto adicional no certificado

## Verificação

Cada certificado tem um código único e QR. Qualquer pessoa pode verificar um certificado:

1. Visite `seucampus.learnbase.lat/verify/CODIGO`
2. Ou escaneie o código QR

A página de verificação mostra se o certificado é válido.

## Enviar por email

Os certificados podem ser enviados para o email do estudante:

1. Vá para o certificado do estudante
2. Clique em **Enviar por Email**

## Regenerar

Se precisar atualizar um certificado (ex: o estudante mudou o nome):

1. Vá para o certificado
2. Clique em **Regenerar**

O código de verificação permanece o mesmo.
