## Como funciona o progresso

O Learnbase rastreia automaticamente o progresso dos estudantes:

- **Vídeos**: Progresso salvo baseado no tempo de visualização
- **Documentos**: Marcado como completo quando aberto
- **Quizzes**: Marcado como completo quando enviado

## Estados do progresso

Cada item de conteúdo pode estar:

- **Não iniciado**: O estudante ainda não acessou
- **Em progresso**: O estudante começou mas não terminou
- **Completo**: O estudante terminou o item

## Completar um curso

Um curso é completado quando todos os itens em todos os módulos estão marcados como completos.

Quando um estudante completa um curso:
- O progresso mostra 100%
- Um certificado é gerado (se habilitado)
- A conclusão é registrada

## Ver progresso

### Como administrador

Vá para **Gestão > Inscrições** e clique em qualquer inscrição para ver:

- Porcentagem de progresso geral do curso
- Progresso por módulo
- Status de cada item

### Como estudante

Os estudantes veem seu progresso:

- Na barra lateral do player do curso
- No dashboard "Meus Cursos"
- Através da barra de progresso em cada card de curso

## Continuar aprendendo

Os estudantes podem continuar de onde pararam. O sistema lembra:

- Última posição do vídeo assistido
- Último módulo acessado
- Itens incompletos
