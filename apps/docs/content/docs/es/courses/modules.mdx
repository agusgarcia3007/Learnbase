Los módulos son secciones que agrupan contenido relacionado dentro de un curso.

## Crear un módulo

1. Ve a **Contenido > Módulos**
2. Haz clic en **Nuevo Módulo**
3. Ingresa un título y descripción
4. Guarda

## Agregar contenido a un módulo

Cada módulo puede contener:

- **Videos**: Lecciones grabadas
- **Documentos**: PDFs, guías, recursos
- **Quizzes**: Evaluaciones para medir conocimiento

Para agregar contenido:

1. Abre el módulo
2. Haz clic en **Agregar Item**
3. Selecciona el tipo de contenido
4. Elige de tu biblioteca o crea uno nuevo

## Organizar contenido

- Arrastra y suelta para reordenar items dentro de un módulo
- Marca items como "Preview" para que usuarios no inscritos puedan verlos
- El orden del contenido determina la secuencia de aprendizaje

## Agregar módulos a un curso

1. Ve a tu curso
2. En la sección **Módulos**, haz clic en **Agregar Módulo**
3. Selecciona módulos de tu biblioteca
4. Arrastra para reordenar

Un curso puede tener múltiples módulos, y un módulo puede reutilizarse en diferentes cursos.
